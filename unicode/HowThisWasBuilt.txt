 Build will create .dat file for unicode data.
      runConfigureICU Linux --with-data-packaging=archive
      // This means that in main() we need to call u_setDataDirectory()
      //  with the path to the .dat file.
      // The icupkg tool can then be used to modify the endianness of the file per platform.
      
To configure and build:
    1. From <icu>/source: Run "chmod +x runConfigureICU configure install-sh"
    2. Run <icu>/source/runConfigureICU --help to find platform keyword to use. (e.g., linux)
    3. CD to the directory we wish to have the build results in then runConfigureICU
    4. Run runConfigure as given below: (but modify for version and platform)
    5. Build with "make". It must be gnu make
    6. Test the build with "make check"
    
    cd <>/unicode/builds/linux
    ../../release-49-1-1/source/runConfigureICU Linux --with-data-packaging=archive --enable-static --disable-shared --prefix=/home/bruce/proteus/unicode/install
    make check
    make
    make check
    make install
    
    
    --------
    "To locate data: ICU will look in /home/bruce/proteus/unicode/install/share/icu/49.1.1 which is the installation location. Call u_setDataDirectory() or use the ICU_DATA environment variable to override."
